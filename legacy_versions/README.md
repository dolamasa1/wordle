# 🏛️ Legacy Versions

This directory contains older, standalone versions of the Wordle game. These are preserved for reference but are not the primary implementation.

## 📄 Files
- **`wordle.html`**: Initial prototype.
- **`wordle2.html`**: Second iteration.
- **`wordle4.html`**: A standalone version with inline CSS and JavaScript, useful for offline testing or single-file distribution.

The main, refactored version of the game is located in the root `index.html`.
